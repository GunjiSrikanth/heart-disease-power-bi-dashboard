# Heart Disease Analytics Dashboard

This project analyzes heart disease patient data to identify risk patterns and trends using Power BI.

## Tools Used
- Microsoft Power BI
- Excel
- Data Visualization

## Dashboard Insights
- Total Patients: 1025
- Heart Disease Patients: 526
- Healthy Patients: 499
- Heart Disease Rate: 51%
- Average Age: 54.43

## Key Features
- Age group analysis of patients
- Heart disease distribution by gender
- Cholesterol and heart rate correlation
- Interactive filtering
